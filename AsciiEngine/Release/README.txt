Double click exe to run demo.

The demo starts in the first state of three total states. Press keys 1, 2 or 3 to go to their respective states.

State 1:
Loads all game entities from file using the deserialization functionality. Saves the state of all objects upon state end to disk.

State 2:
Deserialize the saved data from state 1.

State 3:
Example (unfinished) map editor. Left click to place Tile objects. This demonstrates the new button class by detecting clicks onto the drawing area via a single button.
